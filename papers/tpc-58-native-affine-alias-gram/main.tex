\medskip
\noindent\textbf{Convention.}
One nonzero physical shift \(h_0\), one direct residue/orbit slice, and one
side of the inherited terminal system are fixed unless a left/right direct
sum is displayed.  All Hilbert norms use raw counting measure.  The cutoff
profile is written as
\(\Lambda_X(n)=(\log n)\Phi_0(n/X)\); this is the normalized version of the
same shaped terminal weight denoted \(\Lambda\) in the inherited papers and
is not the von Mangoldt function.  Every label multiplicity is taken on a
fixed baseline carrier after identically zero coordinates, but not
coefficient-dependent cancellations, have been removed.
