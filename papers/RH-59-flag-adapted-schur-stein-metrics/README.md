# RH-59: flag-adapted Schur--Stein metrics

This directory contains the fifty-ninth RH-layer paper:

> *Flag-Adapted Schur--Stein Metrics for Directional Hardy Packets: Exact
> Dissipation and the Outer-Packet Bottleneck*

## Main result

For every stable upper-triangular Schur partition

```text
T = (T_ij),   D_i = T_ii,
```

RH-59 constructs canonical local metrics

```text
H_i - D_i^* H_i D_i = I
```

and proves that positive block scalings make

```text
P = direct_sum_i s_i^2 H_i,
R = P - T^* P T
```

strictly positive. Thus every stable Schur flag admits a compatible block
diagonal Lyapunov metric. This is an exact finite-dimensional existence
theorem, not a small-noise uniform estimate.

For observation `Y`, put

```text
kappa = ||Y R^(-1/2)||_2^2.
```

Then `kappa P` is a positive observability Stein supersolution. Applied on
the invariant prefix generated by source packet `X_j`, it gives

```text
e_j^2 <= kappa_j ||s_j H_j^(1/2) X_j||_F^2.
```

This keeps the complete matrix dissipation instead of replacing every Schur
path by a separate norm.

## Five-scale result

At `sigma=0.01`, the exact left/right Hardy energies are `1.4681` and
`1.7603`. The packetwise flag-metric absolute uppers are `19.2196` and
`12.1382`, versus the RH-58 scalar-path uppers `1922.40` and `380.40`.
Thus exact dissipation improves those sufficient bounds by factors about
`100.0` and `31.3`.

The improvement is not a Stage A1 closure. The fitted growth exponents of
the metric uppers are `0.898` and `0.763`. The smallest-scale outer-packet
uppers are `13.7336` and `6.7965`, while the exact outer-packet energies are
`0.5262` and `1.1310`. The remaining loss is therefore localized rather
than combinatorial.

## Theoretical boundary

Analytic finite-dimensional results:

- canonical local Schur-block Lyapunov metrics;
- flag-compatible block diagonal metric existence;
- exact-dissipation packetwise Stein supersolutions;
- comparison with one-number contraction bounds;
- a two-scalar-block endpoint tradeoff showing why extreme hierarchy cannot
  remove the outer-to-central observation cost.

Computer evidence:

- all production Schur forms, optimized weights, and dense Lyapunov solves
  are binary64 diagnostics;
- the 256-bit Arb certificate covers one two-scalar-block model only;
- no noise-uniform or polylogarithmic packet budget is proved.

Stage A1, unconditional Stage A4, a self-adjoint Hilbert--Polya operator,
`T log T` counting law, arithmetic prime-power trace formula, zeta-zero
identity, and Riemann-hypothesis conclusion remain open.
