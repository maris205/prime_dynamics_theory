\noindent\textbf{Keywords:} coherent terminal-prime square; phase-sector
selection; residual fiber; fixed physical shift; collision kernel; parity
boundary.

\medskip
\noindent\textbf{Convention.}
One nonzero physical shift \(h_0\), one left source system, and one direct
residue/orbit slice are fixed.  All domain and output spaces use the raw
counting-measure atomic normalization inherited from TPC--53.  The shaped
terminal weight
\(\Lambda(n)=(\log n)\Phi_0(n/X)\) is displayed in the operator and is not
the von Mangoldt function.  A selector constructed from the realized
terminal phases is called \emph{adaptive}; this word is never suppressed
when the selector is used.
