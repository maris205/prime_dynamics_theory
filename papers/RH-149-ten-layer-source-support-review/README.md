# RH-149: Ten-Layer Source-to-Support Review

RH-140--RH-149 turns the previous directional maze into a three-interface
frontier.  The exact conditional theorem is complete; what remains is to
supply, on one common all-level path:

1. `E-source`: normalized source and packet enclosures;
2. `E-update`: branch-stable propagation into the outward tail/base assembly;
3. `E-cocycle`: a delayed correlated support cocycle with bounded drawdown.

This set is inclusion-minimal for the present architecture: all three imply
eventual positive directional support, and an omission witness exists for
each one.  The finite evidence is strong—10/10 packets, 360/360 branch
margins, 330/330 outward residual pairs, 28/30 positive tubes, and an 18/18
clean suffix—but it is not an all-level proof.

The roadmap promotes `E-update` to the next priority because it is the finite
bridge between two already-certified endpoints and has the highest immediate
falsification value.  The unsigned projective-summability route is retained
only as a fallback; the correlated support cocycle is the primary asymptotic
route.
