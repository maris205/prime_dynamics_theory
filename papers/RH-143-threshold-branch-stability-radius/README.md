# RH-143: Threshold-Branch Stability Radius

For a relative singular-value threshold `tau`, this paper gives the exact
open operator radius preserving the clipped adaptive width.  Interior widths
must protect both the last retained and first omitted ratios; minimum and
maximum clipping remove the irrelevant side.  The radius vanishes sharply at
a threshold contact.

The projected-cross map `K(A,P)=(I-P)AP` also satisfies

`||K(A,P)-K(B,Q)|| <= ||A-B|| + 2 ||B|| ||P-Q||`.

All 360 archived RH-96 width decisions have positive nominal branch radius
and dominate a conservative local fp64 backward proxy.  At the primary
threshold the smallest relative branch radius is only `4.3449e-9`; therefore
the result is not yet a source-to-update interval certificate.
