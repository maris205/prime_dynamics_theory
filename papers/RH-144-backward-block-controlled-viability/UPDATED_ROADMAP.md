# Roadmap after RH-144

The finite tail-control question is now solved at the scalar candidate-family
level: backward kernels distinguish true control obstructions from poor
forward choices.  The two failures are unavoidable only at a coarse finite
birth.  The next step is to formalize delayed start and finite-prefix
invariance, then audit whether every finer anchor lies beyond the bad birth.

For an all-level proof, one still needs a repeating family of blocks whose
backward kernels contain one common radius bounded away from zero.
