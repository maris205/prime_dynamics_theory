# RH-98: projector-Lipschitz propagation barrier

This directory contains the ninety-eighth RH-layer paper:

> *Projector-Lipschitz Propagation Envelopes and the Failure of Universal Unit Tail Propagation*

## Main results

For rank-r packets U and V and endpoint Gramian G,

    |E_G(U) - E_G(V)| <= ||G||_F ||UU* - VV*||_F.

If a local adaptive Ritz packet has capture loss epsilon relative to the
leading rank-r packet of a compressed matrix with gap gamma, then

    ||P_adaptive - P_full||_F <= sqrt(2 epsilon / gamma).

If the future full block is projector-Lipschitz with constant L, these combine
into the conditional endpoint envelope

    endpoint effect <= ||G_N||_F L sqrt(2 epsilon / gamma).

## Positive audit

Across all 38 omissions generated by the three RH-96 thresholds:

- all local gap-to-projector bounds close;
- all endpoint Gram/projector bounds close;
- all conditional secant envelopes close;
- tail amplification is at most `1 + 4e-16` numerically;
- projector secant amplification nevertheless reaches 8.85112.

## Negative theorem boundary

Universal unit tail propagation is false. A stored 6-by-6, trace-one,
strictly positive two-step Ritz example has validated local loss
`5.28909e-4` and endpoint effect `2.35338e-2`, giving amplification greater
than 44.4949.

Thus the production unit behavior cannot be assumed. RH-99 should derive a
neighborhood-certified differential/projector Lipschitz constant for the
actual separated Ritz maps.
