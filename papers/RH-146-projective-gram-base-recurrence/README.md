# RH-146: Projective Gram Base Recurrence

For positive definite Gramians, Hilbert projective distance gives the sharp
one-step lower

`a(H) >= exp(-d_proj(G,H)/2) a(G)`

for the normalized base `a(G)=sqrt(lambda_min(G)/lambda_max(G))`.  Summable
projective variation is therefore sufficient for a positive base liminf.
Uniformly bounded variation is not sufficient: `G_n=diag(exp(n),1)` has
unit step distance and `a(G_n)->0`.

The physical polar-gauge audit covers all 330 RH-135 transitions.  Every
inequality passes, but the accumulated lower is extremely conservative and
the observed projective increments do not trend toward zero across the five
frozen anchors.  The theorem supplies a rigorous new route to the base
packet; this finite archive does not support, and does not disprove, the
required all-level summability hypothesis.
