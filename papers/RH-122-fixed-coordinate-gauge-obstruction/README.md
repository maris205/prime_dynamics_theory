# RH-122: fixed-coordinate gauge obstruction

RH-122 proves that the gauge in RH-120 is logically necessary.  There is a
four-dimensional family with identical source and target relative constants
`gamma`, related by an exact coordinate swap, for which the best
fixed-coordinate transfer factor is exactly `1/epsilon` and therefore
diverges.  The swapping gauge restores factor one.

The full 512-point audit follows the exact slope `-1` with zero failures.
This closes the no-gauge shortcut; it does not prove that a uniformly good
physical gauge exists.
