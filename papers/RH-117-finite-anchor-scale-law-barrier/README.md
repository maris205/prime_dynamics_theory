# RH-117: finite-anchor scale-law barrier

RH-117 aligns capacity, exterior concentration, fourth-volume, fourth-mode,
and adaptive-depth envelopes across the five archived scales.  The power-law
fits are descriptive only: the median-capacity exponent is `-4.496`, but its
leave-one-out values range from `-5.836` to `-2.621`; the worst fitted
envelope has a multiplicative residual above `141`.

The exact result is an asymptotic nonidentifiability theorem.  Any finite set
of positive anchors admits a smooth positive continuation with an arbitrary
positive germ near zero.  A bounded version preserves an interval such as
`(0,1)`.  Applied to the five median-capacity anchors, three bounded smooth
continuations match every anchor while tending to `0`, `1/2`, and `1`.

Because capacity and relative singular modes are bounded by one, their
observed negative fitted exponents are necessarily pre-asymptotic.  An
all-level theorem must come from a new physical inequality, not finite-anchor
extrapolation.
