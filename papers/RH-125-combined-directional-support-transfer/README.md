# RH-125: combined directional support transfer

defect-stable gamma upper, four-frame determinant transfer, leading-scale
upper, and capacity upper.  The resulting one-step lower is

```text
[a(1-eta)]^2 |det S| /(ell^4 c)
  * (1-gamma_upper)_+^4 * V/(L^4 C).
```

five-scale chains all finish above `1e-8` (23 above `1e-6`, 21 above
`1e-4`).  Because RH-121 added a conditioning floor to the recent Gram,
